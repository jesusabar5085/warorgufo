## ⚙️ Understanding the Pipeline

The application relies on a background process that manages document intake. This pipeline handles several technical tasks to keep the information accurate. 

First, the system scans incoming data from government archives. It checks the files for OCR, or optical character recognition, errors. If a document appears as just a photo, the tool translates the images into text. Then, the system cross-references these details with existing records from the AARO and FBI databases. This ensures that the story you read matches the contents of the official file.
